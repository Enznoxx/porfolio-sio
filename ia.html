Projet d'IA Local

Présentation du projet

Ce projet a été réalisé pour le fun dans le but de mettre en place une IA en local avec une interface graphique. Il comprend également la mise en place d'un service DNS pour le réseau local.

But du projet

L'objectif de ce projet est de déployer une intelligence artificielle locale accessible via une interface graphique. Cette IA peut être utilisée pour des tâches variées et est conçue pour être accessible facilement. J'ai également mis en place un service DNS pour faciliter l'accès aux différents services locaux de l'IA.

Contexte

Ce projet est un projet personnel dans lequel j'ai exploré l'implémentation d'une IA locale et d'une interface graphique via Open-WebUI. En plus de l'IA, j'ai mis en place un service DNS afin d'améliorer l'efficacité et la gestion des services réseau internes pour une utilisation simplifiée.

Mes tâches pendant le projet

  • Mise en place d'une IA en local
  • Création d'une interface graphique avec Open-WebUI
  • Configuration d'un service DNS pour le réseau local
